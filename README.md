# contact-form-7-wpml

## Requirements

- WordPress
- WPML 3.3+
- Contact Form 7

## Installation

1. Clone the repo
2. Zip the files
3. Upload to your WordPress as a plugin

## Usage

- You can find the labels to be translated in the `String Translation` menu under `WPML`
